# Metoda Nelder-Mead
Metoda Nelder-Mead je algoritem za minimizacijo nelinearne funkcije več spremenljivk. Pri svojem delovanju uporablja oglišča simpleksa in njihove funkcijske vrednosti. Program je bil napisan v sklopu diplomskega dela Metoda Nelder-Mead na prvi stopnji interdisciplinarnega programra računalništvo in matematika na Fakulteti za računalništvo in informatiko ter Fakulteti za matematiko in fiziko Univerze v Ljubljani. 
## Uporaba
Za delovanje program potrebuje ```python3```.
Pred poganjanjem programa lahko v datoteki ```neldermead.py``` spremenimo nekatere parametre, kot so:
- ```x0```: začetni približek,
- ```function```: izbira funkcije, ki jo želimo testirati,
- ```isGreedy```: uporaba požrešnega razširjanja,
- ```midPrint```: izpis vmesnih približkov.

Program poženemo prek ukazne vrstice s klicem ```python3 neldermead.py```.
